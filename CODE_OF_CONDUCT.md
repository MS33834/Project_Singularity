# 行为准则

Project Singularity 欢迎所有人参与。我们希望通过友好、开放、尊重的协作环境，让每个人都能安心地贡献和学习。

## 我们的承诺

- 对所有人保持尊重，不论其经验水平、性别、身份、国籍或背景。
- 以建设性方式讨论技术和创意问题。
- 耐心帮助新手，包容不同的技术背景。
- 专注于把项目做得更好。

## 不可接受的行为

- 发表仇恨、歧视、骚扰或侮辱性言论。
- 发布垃圾信息、广告或与项目无关的内容。
- 泄露他人隐私或敏感信息。
- 故意破坏协作氛围。

## 执行

项目维护者有权删除、编辑或拒绝不符合本准则的内容。严重违规者可能被暂时或永久禁止参与项目。

## 适用范围

本准则适用于本项目的所有公共空间，包括 Issue、Pull Request、讨论区以及任何以 Project Singularity 名义组织的活动。

## 联系我们

如果你遇到或目睹违规行为，请通过 Issue 私下联系维护者。

---

> 本行为准则参考 [Contributor Covenant](https://www.contributor-covenant.org/) 制定。
