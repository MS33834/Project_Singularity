# Project Singularity — 团队角色与分工

## 核心团队（建议配置）

| 角色 | 人数 | 职责 | 关键技能 |
|------|------|------|----------|
| **项目制片人 / PM** | 1 | 统筹进度、预算、风险、对外合作 | 项目管理、沟通、预算控制 |
| **导演 / 创意总监** | 1 | 把控艺术方向、叙事节奏、最终成片质量 | 影视叙事、审美、剪辑思维 |
| **编剧 / 策划** | 1 | 剧本、世界观、角色圣经、分镜脚本 | 科幻叙事、提示词工程 |
| **AI 美术操作员** | 1-2 | Flux 出图、角色一致性控制、关键帧生产 | ComfyUI、IPAdapter、PuLID、提示词 |
| **AI 视频操作员** | 1-2 | Wan2.2 / 可灵视频生成、参数调优 | ComfyUI、I2V、视频生成参数 |
| **技术负责人** | 1 | 本地环境部署、模型管理、工作流封装 | Python、CUDA、ComfyUI、Linux |
| **后期剪辑 / 调色** | 1 | 剪辑、调色、画质增强 | 达芬奇 / 剪映 / Topaz / AE |
| **声音设计 / 作曲** | 1 | 配音、音效、配乐、混音 | ElevenLabs、Suno、音频后期 |
| **运营 / 发行** | 1 | 发布、社区教程、反馈收集 | 社交媒体、内容运营 |

## 外部资源

| 资源类型 | 建议方案 | 用途 |
|----------|----------|------|
| 云端视频生成 | 可灵 2.5 Turbo API | 复杂镜头 / 首尾帧约束 |
| 云端算力备选 | RunPod / Vast.ai / AutoDL | 本地 4090 不足时弹性扩容 |
| 模型与 LoRA | Civitai / HuggingFace | 风格模型与角色模型补充 |
| 音效素材 | Epidemic Sound / 爱给网 / 自建库 | 环境音与 Foley |

## 沟通机制

- 主协作工具：飞书 / Notion / 腾讯文档
- 资产管理：NAS / 云盘 + 统一命名规范
- 版本控制：Git LFS（工作流 JSON、参考图）+ 网盘（大体积视频）
