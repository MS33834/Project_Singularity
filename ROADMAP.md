# Project Singularity — 路线图

> 本文件列出项目的短期、中期和长期计划。

---

## Phase 1：完成《奇点回响》示例片

**时间**：当前 — 未来 4 周

### 目标

- 完成《奇点回响》全部 24 个镜头的视频生成。
- 完成配音、配乐、调色、4K 超分。
- 输出最终成片并发布。

### 关键任务
- [ ] 在 RTX 4090 工作站上完成 ComfyUI 部署。
- [ ] 生成 29 张关键帧并通过角色一致性盲测。
- [ ] 生成 19 个 Wan2.2 标准镜头。
- [ ] 生成 5 个可灵复杂镜头。
- [ ] 完成达芬奇粗剪、调色、混音。
- [ ] 输出 4K 母版并上传发布。

---

## Phase 2：工作流增强（未来 1–2 个月）

### 目标
- 把流程打磨到可以一键复用到其他短片项目。

### 关键任务
- [ ] 把 ComfyUI 工作流进一步参数化，支持非程序员通过修改 YAML 配置生成。
- [ ] 引入 Provider 评分机制，在云端 API 与本地模型之间自动选择。
- [ ] 完善 `render_queue.py`，支持优先级、断点续跑、失败重试。
- [ ] 补充更多失败案例与修复记录。

---

## Phase 3：社区化（未来 3–6 个月）

### 目标
- 让项目对更多人可用、可贡献。

### 关键任务
- [ ] 提供 WebUI 或 Gradio 界面，降低使用门槛。
- [ ] 支持更多视频生成模型（HunyuanVideo、LTX-Video、CogVideoX 等）。
- [ ] 增加多语言文档（英文 README、日文等）。
- [ ] 建立用户案例展示区。
- [ ] 发布技术博客与视频教程。

---

## 长期愿景

让 Project Singularity 成为 AIGC 短片制作的「脚手架」：任何人只要有一个好的故事想法，就能按照这套流程在几周内产出自己的 AIGC 短片。
