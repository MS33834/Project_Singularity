# 示例目录

> 这里存放可以直接参考或运行的示例文件，方便快速理解 Project Singularity 的使用方式。

---

## 文件说明

| 文件/目录 | 用途 |
|------|------|
| `奇点回响/` | 完整案例研究：计划书、制作日志、角色圣经、镜头进度表 |
| `character_prompts.md` | 艾娃角色一致性提示词示例 |
| `storyboard_sample.md` | 前 3 个镜头的简化分镜示例 |
| `comfyui_api_payload.json` | 调用 ComfyUI API 的简化载荷示例 |
| `env.example` | 最小环境变量配置模板 |

---

## 快速参考

1. 先看 `奇点回响/README.md`，了解一个完整案例是怎么从头到尾被规划和跟踪的；
2. 再看 `character_prompts.md`，理解角色锚点怎么写；
3. 再看 `storyboard_sample.md`，理解分镜格式；
4. 把 `env.example` 复制为 `.env` 并填入你的 API 密钥；
5. 参考 `comfyui_api_payload.json` 改造为你自己的 ComfyUI 工作流 API 调用。
