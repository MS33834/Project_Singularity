AIGC video production pipeline

From script to 4K master, repeatable.

Project Singularity is a workflow template for making 3–5 minute AIGC short films. It includes scripts, storyboards, ComfyUI workflows, automation scripts, and a worked example: the sci-fi micro-short Echo of the Singularity.

View on GitHub View on GitCode

What this project is

Making an AIGC video is easy for a five-second clip. At three to five minutes, things fall apart: characters look different shot to shot, prompts drift, parameters get lost, and teams step on each other’s work.

This repository bundles the documents, scripts, and workflows we use to keep production sane. It is not a magic button. It is a structured way to plan, generate, check, and deliver.

Locked characters

Flux.1 Kontext + IPAdapter keep the lead character consistent across all 24 shots.

Hybrid generation

Standard shots run locally on Wan2.2 I2V; complex moves use Kling 2.5 Turbo with start/end frames.

Tracked production

Shot tracker, render queue, daily briefs, and QA checklists keep everyone aligned.

Full post pipeline

DaVinci edit and color, ElevenLabs dialogue, Suno music, Topaz 4K upscale.

Quick start

  1. Clone the repository and install Python dependencies with make setup.
  2. Read AIGC_Experience_Chain.md to understand the full workflow.
  3. Explore the examples/奇点回响/ case study for a worked production plan.
  4. Run make check to verify the project structure.
  5. Open the ComfyUI workflows in 03_Workflows/ and adapt them to your project.
